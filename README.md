# sc-form

A static HTML/CSS/JS practice project: a "Student Council Application Form" with student details (name, registration number, department, CGPA, hosteler/day-scholar radio, a photo file input), plus application details (position applying for, and four free-text questions). On submit, JavaScript just prevents the page reload and shows a browser `alert("Thanks for filling the form!")`. There is no backend and no data is saved or sent anywhere.

This appears to be a variant/second pass of the same "Student Council application form" idea as the separate `forms` repo, with different fields and layout.

## Tech Stack

- HTML, CSS, vanilla JavaScript
- No frameworks, no backend, no build tooling

## Structure

```
index.html   - form markup
style.css    - styling
script.js    - submit-event handling (alert only, client-side)
SC.png       - logo/image asset
```

Coursework/practice exercise, not a deployed or production form.
